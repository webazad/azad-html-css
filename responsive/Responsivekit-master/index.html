Responsivekit

Version 1.0

About Responsivekit?

Responsivekit was born with a more deeply support for mobile world in mind, trying to keep at minumin javascript and css size. With a conscious use of css media queries, I implemented a mobile first approach in wich a mobile browser is not constrained to apply a desktop browser style to then overwrite it with a mobile one.

Features

Responsivekit is built with features:

  • A Responsive Grid: Responsivekit is based on the famous 960 grid, but it's not limited to 960 pixels! It provides also a 1200 pixels grid for bigger screens. Responsivekit it's not fixed on it's size: it rescales down step by step adapting to smaller sizes till the mobile one, when a fully fluid full screen layout it's used.
  • Slick and fast to use: Responsivekit is based on modern technologies and it is well structured. Starting up requires only download just a bunch of files, linking it in your html, use css classes provided and it's done!
  • Responsive data tables: Responsivekit offers a standing feature compared to other boilerplates: responsive tables! Using a mix of javascript and css media queries, Responsivekit changes tables aspect on the fly adpting to mobile sizes.
